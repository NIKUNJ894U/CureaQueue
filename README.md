# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/f708302e-fea6-4d13-ad5e-0de0997f9ff3

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

### 3. 🖥️ Lobby TV Dashboard View (Public Display Mode)

- **What it does:** A clean, spacious, high-contrast dashboard layout designed to be displayed on waiting room monitors or mounted lobby TV sheets.
- **Why it's great:** Displays active token calls with beautiful pulsing animations.
- Shows anonymized patient labels (e.g., Token #12 - John D.) to guarantee **HIPAA/privacy compliance**.
- Auto-translates queue status updates or emergency alerts instantly.
